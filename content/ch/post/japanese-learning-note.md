## 助词

### 「に」

{{< gramma >}} 时间名词 に 动词。{{< /gramma >}}提示动作发生时间。

例：

田中さんは毎日六時半に起きます。田中先生每天六点半起床。

林さんは毎晩働きます。林先生每晚工作。*注：不含具体时间数字不加「に」，星期可加可不加。*

### 「へ」

{{< gramma >}} 地点名词 へ 移动性动词。{{< /gramma >}}

例：

京都へ行きます。我要去京都。

去年の３月、日本へ来ました。我去年3月来的日本。

### 「も」

{{< gramma >}} N1 も N2 です。{{< /gramma >}} 也...

例：

私も学生です。我也是学生。

{{< gramma >}} 疑问词 も 否定表达。{{< /gramma >}} 全部否定。

例：

どこへも行きません。我哪也去不了。

何も食べません。我什么也不吃。

誰もいません。谁都不在。

## 句型

### 判断句肯定形式

{{< gramma >}}名词1 は 名词2 です{{< /gramma >}}

例：あの人は学生です。那个人是学生。

### 判断句否定形式

{{< gramma >}}名词1 は 名词2 ではありません/じゃありません{{< /gramma >}}

例：あの人は学生 ではありません/じゃありません。那个人不是学生。

### 一般疑问句

{{< gramma >}}名词1 は 名词2 ですか{{< /gramma >}}

例：あの人は学生ですか。那个人是学生吗?

### 誰/何方

{{< gramma >}}名词 は 誰/何方 ですか{{< /gramma >}}

例：あの人は{{<kanji 誰 だれ>}}/{{<kanji 何方 どなた>}}ですか。那个人是谁?

### 何歳/(お)いくつ

{{< gramma >}}名词 は 何歳/(お)いくつ ですか{{< /gramma >}}

例：あの人は{{<kanji 何歳 なんさい>}}/おいくつですか。那个人几岁?

### 选择疑问句

{{< gramma >}}名词1ですか、名词2ですか{{< /gramma >}}

例：

\- 私の鞄はこれですか、それですか。我的包是这个还是那个？

\- これです。这个。

### この/その/あの

{{< gramma >}}この/その/あの名词は〜のです{{< /gramma >}} 这/那个～是～的。

例：この手帳は私のです。这个手帐是我的。

### どれ/どの

-- 鈴木さんの机はどれですか。铃木先生的桌子是哪个？  
-- これです。是这个。

### ここ/そこ/あそこ

{{< gramma >}}ここ/そこ/あそこ は 地点名词 です{{< /gramma >}} 这/那里是...。

例：ここは学生の食堂です。这里是学生食堂。

### 物体所在地点

{{< gramma >}}名词 は 地点名词 です{{< /gramma >}} ...在...。

例：電話は二階です。电话在二楼。

### 询问地点

{{< gramma >}}名词 は どこ/何方 ですか{{< /gramma >}} ...在哪。「{{<kanji 何方 どちら>}}」为敬语形式。

例：お手洗いはどこですか。洗手间在哪？

### 询问时间

{{< gramma >}}今何時(何分)ですか{{< /gramma >}} ...现在几点几分?

例：ニューヨークは今何時ですか。纽约现在几点？
