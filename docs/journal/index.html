刘畅的日记

2022-01-02 深圳 晴

  • coto日语课，主要进行第六课的复习，没有太多新知识。
  • 阅读Kubernates in Action，随书敲命令并记录。
  • 晚餐与褚尝试了壹方城的鸟笑(とりわらい)，新店开张，用料自然舍得，期待以后表现。
  • 晚餐后与褚散步，发现宝安政府对面，过了平日跑步的操场，即是宝安图书馆，少年宫，剧院，欢乐港湾公园。
  • 于宝安图书馆借阅了《洞山指月》，褚借《傅雷家书》，36天内归还。
  • 与褚同看《短剧开始啦(コントが始まる)》，配冰牛奶一杯。
