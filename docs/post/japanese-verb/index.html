一类动词/五段动词

 以「う」段假名结尾的动词。 

特殊一类动词: 嘲あざけ る、焦あせ る、要い る、煎い る、返かえ る、帰かえ る、限かぎ る、切き る、覆くつがえ る、蹴け る、遮さえぎ る、茂しげ る、湿しめ る、知し る、滑すべ る、散ち る、照て る、握にぎ る、練ね る、罵ののし る、入はい る、走はし る、減へ る、参まい る、混ま じる、漲みなぎ る

「ます」型

 「う」段->「い」段＋ます 

「た」型

  •  以「う/つ/る」结尾的 → 「う/つ/る」 + 「った」 
  •  以「む/ぬ/ぶ」结尾的 → 「む/ぬ/ぶ」 + 「んだ」 
  •  以「く」结尾的 → 「く」 + 「いた」 
  •  以「ぐ」结尾的 → 「く」 + 「いだ」 
  •  以「す」结尾的 → 「す」 + 「した」 

「て」型

  •  以「う/つ/る」结尾的 → 「う/つ/る」 + 「って」 
  •  以「む/ぬ/ぶ」结尾的 → 「む/ぬ/ぶ」 + 「んで」 
  •  以「く」结尾的 → 「く」 + 「いて」，特例：「行く」→ 「行って」 
  •  以「ぐ」结尾的 → 「く」 + 「いで」 
  •  以「す」结尾的 → 「す」 + 「して」 

二类动词/一段动词

 以「る」结尾的动词，且「る」之前的假名为「え」「い」段。 

「ます」型

 「る」 + 「ます」 

「た」型

 「る」 + 「た」 

「て」型

 「る」 + 「て」 

常用二类动词变形表

三类动词

 「来る」/「する」/名词性动词 +「する」。 

「ます」型

  •  「来る」→「来ます」 
  •  「する」→「します」 
  •  名词性动词 +「する」→ 名词性动词 +「します」 

「た」型

  •  「来る」→「来た」 
  •  「する」→「した」 
  •  名词性动词 +「する」→ 名词性动词 +「した」 

「て」型

  •  「来る」→「来て」 
  •  「する」→「して」 
  •  名词性动词 +「する」→ 名词性动词 +「して」
