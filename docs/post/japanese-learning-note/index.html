本文记录日文学习过程中整理的语法。

助词

「に」

  时间名词 に 动词。  提示动作发生时间。

例：

田中さんは毎日六時半に起きます。田中先生每天六点半起床。

林さんは毎晩働きます。林先生每晚工作。注：不含具体时间数字不加「に」，星期可加可不加。

「へ」

  地点名词 へ 移动性动词。 

例：

京都へ行きます。我要去京都。

去年の３月、日本へ来ました。我去年3月来的日本。

「も」

  N1 も N2 です。  也…

例：

私も学生です。我也是学生。

  疑问词 も 否定表达。  全部否定。

例：

どこへも行きません。我哪也去不了。

何も食べません。我什么也不吃。

誰もいません。谁都不在。

动词

动词分类

  • 一类动词/五段动词

     以「う」段假名结尾的动词。 

    例如：会あ う、書か く、話はな す、立た つ、死し ぬ、読よ む、泳およ ぐ、呼よ ぶ。

    特殊一类动词: 嘲あざけ る、焦あせ る、要い る、煎い る、返かえ る、帰かえ る、限かぎ る、切き る、覆くつがえ る、蹴け る、遮さえぎ る、茂しげ る、湿しめ る、知し る、滑すべ る、散ち る、照て る、握にぎ る、練ね る、罵ののし る、入はい る、走はし る、減へ る、参まい る、混ま じる、漲みなぎ る

  • 二类动词/一段动词

     以「る」结尾的动词，且「る」之前的假名为「え」「い」段。 

    例如：起お きる、寝る、食べる。

  • 三类动词

     「来る」/「する」/名词性动词 +「する」。 

    例如：勉強べんきょう する。

「ます」形

  • 一类动词

    「う」段 →「い」段 + ます

  • 二类动词

    「る」+ ます

  • 三类动词

    「来る」→「来ます」

    「する」→「します」

    名词性动词 +「する」→ 名词性动词 +「します」

「て」形

  • 一类动词

    以「う/つ/る」结尾的 → 「う/つ/る」 + 「って」

    以「む/ぬ/ぶ」结尾的 → 「む/ぬ/ぶ」 + 「んで」

    以「く」结尾的 → 「く」 + 「いて」，特例：「行く」→ 「行って」

    以「ぐ」结尾的 → 「く」 + 「いで」

    以「す」结尾的 → 「す」 + 「して」

  • 二类动词

    「る」 + 「て」

  • 三类动词

    「来く る」→「来き て」

    「する」→「して」

    名词性动词 +「する」→ 名词性动词 +「して」

「た」形

  • 一类动词

    以「う/つ/る」结尾的 → 「う/つ/る」 + 「った」

    以「む/ぬ/ぶ」结尾的 → 「む/ぬ/ぶ」 + 「んだ」

    以「く」结尾的 → 「く」 + 「いた」

    以「ぐ」结尾的 → 「く」 + 「いだ」

    以「す」结尾的 → 「す」 + 「した」

  • 二类动词

    「る」 + 「た」

  • 三类动词

    「来く る」→「来き た」

    「する」→「した」

    名词性动词 +「する」→ 名词性动词 +「した」

「ない」形

  • 一类动词

    「う」段 →「あ」段 + 「ない」

    特例：「う」 → 「わ」

  • 二类动词

    「る」 + 「ない」

  • 三类动词

    「来く る」→「来こ ない」

    「する」→「しない」

    名词性动词 +「する」→ 名词性动词 +「しない」

授受动词

  • あげる

      A は B に 名詞 を あげる。 

  • くれる

      A は B 我/我方的人 に 名詞 を くれる。 

  • もらう

      B は A に/から 名詞 を もらう。 

形容词

形容词分类

  • 一类形容词/「イ」型

      以「い」结尾。 

    例如：大きい、

  • 二类形容词/「ナ」型/形容动词

      以「だ」结尾。 

接尾词

「たち」：～们

  人/动物 + 「たち」 

例：私たち、あなたたち、学生たち、彼ら

「歳」：～岁

  数字 + 「歳」 

特殊：一歳いっさい 、八歳はっさい 、十歳じゅっさい 、二十歳はたち

句型

判断句肯定形式

 名词1 は 名词2 です 

例：あの人は学生です。那个人是学生。

判断句否定形式

 名词1 は 名词2 ではありません/じゃありません 

例：あの人は学生 ではありません/じゃありません。那个人不是学生。

一般疑问句

 名词1 は 名词2 ですか 

例：あの人は学生ですか。那个人是学生吗?

誰/何方

 名词 は 誰/何方 ですか 

例：あの人は誰だれ /何方どなた ですか。那个人是谁?

何歳/(お)いくつ

 名词 は 何歳/(お)いくつ ですか 

例：あの人は何歳なんさい /おいくつですか。那个人几岁?

选择疑问句

 名词1ですか、名词2ですか 

例：

- 私の鞄はこれですか、それですか。我的包是这个还是那个？

- これです。这个。

これ/それ/あれ

この/その/あの

 この/その/あの名词は〜のです  这/那个～是～的。

例：この手帳は私のです。这个手帐是我的。

どれ/どの

– 鈴木さんの机はどれですか。铃木先生的桌子是哪个？
– これです。是这个。

ここ/そこ/あそこ

 ここ/そこ/あそこ は 地点名词 です  这/那里是…。

例：ここは学生の食堂です。这里是学生食堂。

物体所在地点

 名词 は 地点名词 です  …在…。

例：電話は二階です。电话在二楼。

询问地点

 名词 は どこ/何方 ですか  …在哪。「何方どちら 」为敬语形式。

例：お手洗いはどこですか。洗手间在哪？

询问时间

 今何時(何分)ですか  …现在几点几分?

例：ニューヨークは今何時ですか。纽约现在几点？
