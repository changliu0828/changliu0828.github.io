动词

动词分类

  • 一类动词/五段动词

    以「う」段假名结尾的动词。

    例如：会あ う、書か く、話はな す、立た つ、死し ぬ、読よ む、泳およ ぐ、呼よ ぶ。

  • 二类动词/一段动词

    以「る」结尾的动词，且「る」之前的假名为「え」「い」段。

    例如：起お きる、寝る、食べる。

  • 三类动词

    「来る」/「する」/名词性动词 +「する」

    例如：勉強べんきょう する。

「ます」形

  • 一类动词

    「う」段 →「い」段 + ます

  • 二类动词

    「る」+ ます

  • 三类动词

    「来る」→「来ます」

    「する」→「します」

    名词性动词 +「する」→ 名词性动词 +「します」

「て」形

  • 一类动词

    以「う/つ/る」结尾的 → 「う/つ/る」 + 「って」

    以「む/ぬ/ぶ」结尾的 → 「む/ぬ/ぶ」 + 「んで」

    以「く」结尾的 → 「く」 + 「いて」

    以「ぐ」结尾的 → 「く」 + 「いで」

    以「す」结尾的 → 「す」 + 「して」

  • 二类动词

    「る」 + 「て」

  • 三类动词

    「来く る」→「来き て」

    「する」→「して」

    名词性动词 +「する」→ 名词性动词 +「して」

「た」形

  • 一类动词

    以「う/つ/る」结尾的 → 「う/つ/る」 + 「った」

    以「む/ぬ/ぶ」结尾的 → 「む/ぬ/ぶ」 + 「んだ」

    以「く」结尾的 → 「く」 + 「いた」

    以「ぐ」结尾的 → 「く」 + 「いだ」

    以「す」结尾的 → 「す」 + 「した」

  • 二类动词

    「る」 + 「た」

  • 三类动词

    「来く る」→「来き た」

    「する」→「した」

    名词性动词 +「する」→ 名词性动词 +「した」

「ない」形

  • 一类动词

    「う」段 →「あ」段 + 「ない」

    特例：「う」 → 「わ」

  • 二类动词

    「る」 + 「ない」

  • 三类动词

    「来く る」→「来こ ない」

    「する」→「しない」

    名词性动词 +「する」→ 名词性动词 +「しない」
